    QC Reports

    These reports list reasons why a gene trap was not loaded. The load will attempt to load them weekly. Only when the issues are fixed will they be loaded. More detail for a given sequence tag can be found in the curation log above.
    1. Cell Line ID Found in an Existing Allele Symbol or Synonym
    2. Gene Trap Creator not Found in Database
    3. Derivation did not Resolve
    4. Gene Traps with Other Resolving Errors
    5. Sequence Associated with a Different Allele in the Database
    6. Sequence Associated with a Marker in the Database
    7. Sequence Object not in MGI
    8. Sequence Tag Vector End Information Not Found
    9. Sequence Tag Method Not Found
